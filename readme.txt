********Read-Me for Tour Application project***********
To make the application run successfully, prepare the backend first which is a sql server.
In the sql server management studio, run the queries which are in 'tourapplicationqueries.sql' file. It creates the tables required for the application.
Click on the tourapplication folder and click on the 'tourapplication.sln' file to open the project in Visual Studio 2012 or above.
In the solution explorer, make the 'index.aspx' as start page by right clicking on that file.
Run the application by clicking on the play (green) button on the top and play with the application in the browser.
